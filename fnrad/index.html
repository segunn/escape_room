Welcome to this virtual Christmas puzzle hunt

We may not be able to meet in person, but we can still imagine some festive fun together.

I hope you'll enjoy this fictional version of some family Christmas traditions, plus some brand new inventions, in virtual form.

Tips
  • You'll need to solve some puzzles to get to the end, so you might want a pen and paper handy
  • One of the puzzles involves sound, so make sure your speakers are on!
  • This set of puzzles is intentionally a bit harder than the others I've made, so it can be played by a larger group. If it's too hard, feel free to ask for hints!
